# OpenFOMs
본 프로젝트는 제조현장의 다양한 데이터를 표준화된 FOM(FileSet) 구조로 정리하고, 4M(Man, Machine, Material, Method) 기반의 다차원 분석 체계를 통해 생산성과 운영 효율을 정량적으로 분석하는 것을 목적으로 한다. QPR을 포함한 데이터 구조를 기반으로 KPI 중심의 분석 절차를 구현하며, 문제 도출과 원인 분석, 개선 방향 제시까지 이어지는 체계적인 FOM Process를 지원한다. 또한 데이터 전처리부터 분석까지의 흐름을 통합하여 현장 중심의 실질적인 운영 개선과 지속적인 성과 향상을 위한 기반을 제공한다.
